
# ea-transport-api

Backend API for East African transport routes, schedules, alerts, and bookings.

This project provides structured transport data to support mobile and web applications, starting with rail services and expandable to other transport modes across East Africa.

## Features
- Transport stations and routes
- Train schedules (SGR & MGR)
- Service alerts and disruptions
- Passenger bookings
- Admin authentication

## Tech Stack
- FastAPI (Python)
- MongoDB
- RESTful JSON APIs

## Status
Under active development.
